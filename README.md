# Barbershop Landing — Демо-лендинг барбершопа

Демонстрационный проект для портфолио. Лендинг для премиального барбершопа.

## Живое демо

[Открыть сайт](https://burr1t0.github.io/barbershop-landing)

## Что реализовано

- Hero-секция со слоганом и кнопкой записи
- Карточки услуг с ценами
- Блок с мастерами
- Секция преимуществ
- Отзывы клиентов
- Форма записи
- Адаптивная вёрстка (мобильные и десктоп)
- Анимации при скролле
